# FUTURE_FS_02 - Mini CRM System

## 🚀 Project Overview

Mini CRM is a Full Stack Client Lead Management System developed using the MERN Stack.

This application helps businesses manage customer leads efficiently by allowing admins to:

- Add leads
- Track lead status
- Manage follow-ups
- Analyze lead conversions
- View business insights through analytics

---

# 🛠️ Tech Stack

## Frontend

- React.js
- Axios
- React Toastify
- Recharts

## Backend

- Node.js
- Express.js

## Database

- MongoDB
