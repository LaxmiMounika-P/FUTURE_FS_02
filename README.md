# Mini CRM - Client Lead Management System

A full-stack Mini CRM application built using the MERN Stack.

## Features

* Add new leads
* Update lead status
* Delete leads
* Analytics dashboard
* Lead source tracking
* Conversion tracking
* Responsive UI
* Settings panel
* Toast notifications
* Dark mode support

## Tech Stack

Frontend:

* React.js
* Vite
* Axios
* Recharts

Backend:

* Node.js
* Express.js

Database:

* MongoDB Atlas
